# DelphiHack UI

## Prerequisites

**Required**

- Git
- Node.js 14 or LTS
- Yarn
- Keplr Wallet browser extension

## Setup local development

```sh
# install dependencies
yarn install

# copy env file and fill in values
cp .env.example .env

# run development server
yarn dev

# (optional) lint and format project
yarn lint
```
